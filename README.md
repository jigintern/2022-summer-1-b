### 概要
- 自販機の写真を撮影してユーザーが投稿出来るようなフォームを作成する
- 投稿したフォームをデータベースに保存し誰でも閲覧出来るようにする
  - DBには **supabase** を使用した
    - DB上にアップロードするために `POST` を実装
    - DB上のデータを取得するために `GET` を実装
- 地図情報を **Leaflet** を使って表示した
